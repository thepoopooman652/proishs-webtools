CDN Information & Usage Guide

What is this?

This is a simple Content Delivery Network (CDN) used to host and serve files. It has both a public directory for general access and a private, token-protected directory for restricted files.

How to Request Files

You can access files by constructing a URL that points to the api.php script.

Public Files

Files in the /public/ directory can be accessed directly. The URL format is:

api.php?file=[PATH/TO/YOUR/FILE.txt]

Example: To get a file named image.png located in /public/images/, you would use: api.php?file=images/image.png

Private Files

Files in the /private/ directory require an authentication token. You must append the auth parameter to the URL.

api.php?file=[PATH/TO/YOUR/FILE.txt]&auth=[YOUR_AUTH_TOKEN]

Example: To get a file named report.pdf from the root of the private folder, you would use: api.php?file=report.pdf&auth=PROISHCDN-API-KEY_main-S2t...
